(un punto) y mal contestada resta 1p (un punto).

1. ¿Qué diferencia hay entre un error de sintaxis y un error de lógica (falla)?

Los errores de sintaxis son los más fácil de corregir y se muestran cuando el programa se compila, y de esa forma se pueden buscar fácilmente para corregirlos.
Los errores de lógica se deben a que hemos creado nuestro algoritmo en algoritmo en forma errónea y aunque todas las sentencias estén correctas, el programa no 
lleva a cabo lo que deseamos. Estos errores son más difíciles de corregir y la mejor forma de evitar es por medio de un análisis previo adecuado del problema a resolver.

2. ¿Cuáles son los operadores de una expresión lógica?

los operadores logicos son los siguientes : Y = "&&" , O = "| |" , no = "!"

3. La refactorización de código: ¿Agrega/quita funcionalidad al programa?

no, ya que como su significado indica, es un código alternativo tenga el mismo 
comportamiento que el código de referencia. Sin modificar o alterar el comportamiento del codigo.

4. Dentro de un ArrayList: ¿se pueden insertar valores de distinto tipo?

Es una lista de elementos de tamaño dinámico. la lista puede almacenar distintos tipos de valores (int, string , float, etc.)
